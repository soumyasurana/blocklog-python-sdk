# Blocklog Python SDK

**Infrastructure for AI Decision-Making.**

Record, audit, and investigate AI decisions.

[![Python 3.11+](https://img.shields.io/badge/python-3.11+-blue.svg)](https://www.python.org/downloads/)

---

## Installation

### From GitHub

```bash
pip install git+https://github.com/blockloghq/blocklog-python.git
```

---

## Quick Start

Initialize the SDK and record a decision.

```python
from blocklog import init, decision

init(
    api_key="bl_your_api_key",
    debug=True
)

with decision(
    type="BUY",
    asset="TSLA"
) as d:
    d.record_input(price=412)
    d.record_output(action="BUY")
```

---

## What Gets Recorded

For each decision, Blocklog records:

* Decision metadata
* Inputs provided to the decision
* Outputs generated by the decision
* Timestamps
* Associated trace information

This creates an auditable record of AI behavior.
