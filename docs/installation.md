# Installation

## Requirements

Python 3.10 or newer.

## Install

```bash
pip install blocklog
```

## Verify

```python
import blocklog
print(blocklog.__version__)  # 0.2.2
```

## Development

```bash
git clone https://github.com/soumyasurana/blocklog-python.git
cd blocklog-python
pip install -e ".[dev]"
```
